Rubygems & Bundler

There's a complete mirror of Rubygems available with instructions for gem and bundle at:

http://rubygems

Caveat: the rubygems website search isn't working yet.

Rbenv & Ruby Build

Experimenting with new rubies? Install them with ruby-build, which is also the rbenv install command, and use our local cache by setting an environment variable:

RUBY_BUILD_MIRROR_URL=http://railscamp/mirror/ruby-build/ rbenv install 2.1.2

Or you can download them directly.
